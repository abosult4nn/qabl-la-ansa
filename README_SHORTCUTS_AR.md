# ربط «قبل لا أنسى» مع Reminders والساعة في iPhone

هذه النسخة لا تستطيع إنشاء تذكير أو منبّه داخل iOS مباشرة لأنها Web App. الحل العملي هو استخدام تطبيق **Shortcuts** كجسر بين التطبيق وتطبيقات Apple.

## الأسماء المطلوبة للاختصارات

أنشئ اختصارين بهذه الأسماء حرفيًا:

```text
قبل لا أنسى - تذكير
قبل لا أنسى - منبّه
```

التطبيق يرسل لكل اختصار نصًا بصيغة JSON مثل:

```json
{
  "title": "أكلم الكهربائي",
  "due": "الساعة 5",
  "bucket": "البيت",
  "type": "reminder",
  "priority": "normal",
  "source": "قبل لا أنسى"
}
```

## اختصار Reminders

الفكرة:

1. يستقبل **Shortcut Input** كنص.
2. يحوّل النص إلى **Dictionary**.
3. يأخذ قيمة `title` كعنوان التذكير.
4. يأخذ `due` و `bucket` ويضعها في الملاحظات أو يستخدمها لتحديد وقت التذكير.
5. يستخدم إجراء **Add New Reminder / إضافة تذكير جديد** في تطبيق Reminders.

اقتراح عملي للبداية: اجعل الاختصار يسألك عن التاريخ/الوقت إذا كان الوقت المرسل غير واضح.

## اختصار الساعة

الفكرة:

1. يستقبل **Shortcut Input** كنص.
2. يحوّل النص إلى **Dictionary**.
3. يأخذ قيمة `title` كاسم المنبّه.
4. يسألك عن الوقت إذا لم يستطع فهم `due`.
5. يستخدم إجراء **Create Alarm / إنشاء منبّه** أو الإجراء المقابل للساعة في Shortcuts.

## داخل التطبيق

بعد إنشاء الاختصارين، ستظهر داخل كل بطاقة أزرار:

- **Reminders**: يرسل المهمة إلى تطبيق التذكيرات.
- **الساعة**: يرسل المهمة إلى اختصار إنشاء المنبّه.

ملاحظة: أسماء إجراءات Shortcuts قد تظهر بالعربي أو الإنجليزي حسب لغة جهازك.
