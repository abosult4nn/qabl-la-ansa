# قبل لا أنسى — نسخة PWA للآيفون

هذه نسخة أولى تعمل كموقع ويب قابل للإضافة على شاشة iPhone الرئيسية.

## التشغيل السريع
1. ارفع الملفات كما هي إلى أي استضافة Static Hosting مثل GitHub Pages أو Netlify أو Vercel.
2. افتح رابط الموقع من Safari على iPhone.
3. اضغط مشاركة.
4. اختر: إضافة إلى الشاشة الرئيسية.
5. تظهر أيقونة "قبل لا أنسى" كتطبيق.

## ما الذي يعمل الآن؟
- حفظ الأفكار والمهام والتذكيرات داخل الجهاز.
- تقسيم النص العربي إلى عناصر متعددة.
- تصنيف مبدئي: مهمة، تذكير، فكرة، موعد.
- استخراج وقت وصفي مثل: اليوم، بكرة، بعد العصر.
- بحث سريع.
- تعديل وحذف وإنجاز.
- تصدير واستيراد JSON.
- PWA Manifest + Service Worker للعمل بعد التحميل.

## ملاحظة مهمة
هذه النسخة لا تستخدم خادم ولا ذكاء خارجي. الفرز الحالي مبني على قواعد عربية بسيطة. لتحويلها لنسخة أذكى نضيف لاحقًا Backend آمن يتصل بنموذج ذكاء اصطناعي، بدل وضع مفاتيح API داخل المتصفح.

## الملفات
- index.html: التطبيق كاملًا.
- manifest.webmanifest: إعدادات PWA والأيقونة.
- sw.js: التخزين المؤقت للعمل كتطبيق ويب.
- icons/: أيقونات التطبيق.
